# Documentação do Projeto

Esta pasta contém toda a documentação técnica e de design do Framework de Famílias de Estratégias.

- **[Especificação das Famílias de Estratégias (STRATEGY_FAMILIES_SPEC.md)](./STRATEGY_FAMILIES_SPEC.md)**
  - **O quê:** Detalha a lógica, os parâmetros, as variantes e as métricas de cada família de estratégia implementada.
  - **Público-alvo:** Pesquisadores Quantitativos, Traders, Analistas de Risco.

- **[Visão Geral e Design do Sistema (SYSTEM_DESIGN.md)](./SYSTEM_DESIGN.md)**
  - **O quê:** Descreve a arquitetura, os componentes, os fluxos de dados e os princípios de funcionamento do framework. É o manual técnico do *motor*.
  - **Público-alvo:** Desenvolvedores, Arquitetos.

- **[Decisões de Arquitetura (ARCHITECTURE_DECISIONS.md)](./ARCHITECTURE_DECISIONS.md)**
  - **O quê:** Registra o "porquê" por trás das principais escolhas de design, os trade-offs considerados e as alternativas descartadas.
  - **Público-alvo:** Arquitetos, Novos membros da equipe, Revisões futuras.

- **[Runbook de Operações (OPERATIONS_RUNBOOK.md)](./OPERATIONS_RUNBOOK.md)**
  - **O quê:** Guia prático com procedimentos para monitorar o sistema, responder a incidentes e executar tarefas de recuperação.
  - **Público-alvo:** Responsáveis Operacionais, DevOps.
